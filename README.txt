
INTERVIEW MANAGEMENT APPLICATION

Project Overview
The Interview Management Application is a web-based application used to schedule, view, edit, and manage job interviews.
It is built using HTML, CSS, and JavaScript and stores data using the browser’s Local Storage.

Features
- Add interview details
- Edit interview details
- Delete interviews
- Search by candidate name
- Filter interviews by status
- Data persistence using Local Storage
- Success messages for user actions
- Clean and user-friendly UI

Technologies Used
- HTML
- CSS
- JavaScript
- Browser Local Storage

Project Structure
Interview-Management-App/
- index.html
- style.css
- script.js
- README.txt

How to Run the Project
1. Download or clone the project
2. Open index.html in a web browser
3. Start adding interview details
4. Data will be saved automatically in Local Storage

Data Storage
The application uses browser Local Storage.
To view stored data:
Right Click → Inspect → Application → Local Storage
